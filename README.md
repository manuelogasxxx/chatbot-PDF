# UI
La UI puedes encontrarla en el siguiente repositorio:
https://github.com/manuelogasxxx/chatbot-ui-final.git

tambien puedes descargar una versión ligera aquí:
https://drive.google.com/file/d/16AMcHJFdZIEEgreYhUxvgT-rxJKbFFU_/view?usp=sharing


instala los requeriments en un entorno virtual o de forma global con el siguiente comando:

pip install -r requirements.txt

ejecuta la aplicación con el siguiente comando:

streamlit run nombre_de_tu_archivo.py

Para que funcione correctamente tendrás que configurar el backend de la página web para apuntar al servidor. Ve a src/api_client.py y cambia BASE_URL por la dirección de tu servidor local.
